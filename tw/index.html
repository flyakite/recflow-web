RecFlow 怎麼運作的？

extension

1. 安裝 Chrome 外掛

確認您正在使用 Chrome
然後 安裝 RecFlow 工具

fiber_manual_record

2. 錄製測試步驟

點 RecFlow 外掛圖示 進入錄製模式， 然後所有您的點擊和輸入會被依序記錄下來，之後會放到實體的行動裝置上回放同樣的步驟做測試。

devices

3. 選擇行動裝置

免費選擇您想要測試的手機、平板電腦。(我們正陸續增加可以付費使用的其他行動裝置。)

bug_report

4. 得到測試結果

測試跑完後，您會得到 JavaScript 除錯訊息、螢幕截圖和裝置上完整的操作影片等。
